XPERIOR 1.3.0 documentation¶

Welcome!

What is XPERIOR: test tool for the telecommunication branch written in Python programming language developed by Noriba GmbH . Xperior aims on testing of devices connected to PC via VISA interface, such TCPIP, GPIB, USB, ASRL, TELNET, etc. and supports SOAP communication. Application consists of built-in Python interpreter, giving the possibility to use huge amount of Python libraries in script itself. Maintenance of test results with Reporter offers clean overview and coordinates test planning.

Introduction¶

Xperior binds together more stand-alone applications and enables their information’s exchange. Therefore also the name. If you are interested in certain sub-application, please refer direct to it.

Communicator¶

Communicator enables scripts execution, further data handling and direct interacting with built-in interpreter. Scripts can run in Python debugger. Useful functions defined in Xperior Library. Keyboard shortcut: Ctrl + F2.

Reporter¶

Reporter offers clear overview and multiple-users administration of bug reports with access to bug reporting server. Keyboard shortcut: Ctrl + F1.
